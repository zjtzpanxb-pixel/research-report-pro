# Research Report Pro

> 对话题进行全面深入研究和分析，生成标杆级报告，所有数据引用均来自可信官方来源

## 🚀 快速开始

### 安装

```bash
# 克隆或复制到工作区
cd ~/.openclaw/workspace/skills/
# skill 已在此目录
```

### 使用

**方式 1: 对话触发**
```
研究一下中国新能源汽车行业
分析十四五规划对数字经济的影响
```

**方式 2: 命令行**
```bash
cd /Users/pxb/.openclaw/workspace/skills/research-report-pro
python main.py "中国新能源汽车行业"
python main.py "人工智能发展" deep 技术 市场 政策
```

## 📋 功能特性

### 核心能力

- ✅ **多源数据采集** - 官方统计/行业报告/学术论文/权威媒体
- ✅ **可信度验证** - 来源分级（政府/学术/媒体/其他）
- ✅ **深度分析** - PESTEL/SWOT/波特五力等成熟框架
- ✅ **数据可视化** - 图表/趋势/对比分析
- ✅ **引用管理** - 完整的参考文献和数据来源标注

### 研究流程

```
1. 话题解析 → 2. 搜索策略 → 3. 数据采集 → 4. 可信度验证
                                           ↓
8. 质量审查 ← 7. 引用标注 ← 6. 报告撰写 ← 5. 深度分析
```

## ⚙️ 配置

### 可信来源分级

| 分级 | 来源类型 | 示例 |
|------|----------|------|
| **Tier 1** | 政府/官方统计 | gov.cn, stats.gov.cn, 国务院 |
| **Tier 2** | 学术/研究机构 | edu.cn, 知网，社科院 |
| **Tier 3** | 权威媒体 | 新华社，人民日报，财新 |

### 分析框架

- **PESTEL** - 政治/经济/社会/技术/环境/法律
- **SWOT** - 优势/劣势/机会/威胁
- **波特五力** - 行业竞争分析
- **价值链分析** - 产业价值分布

## 🔧 故障排除

### 问题 1：搜索失败
检查网络连接，或尝试简化话题

### 问题 2：来源不足
扩大搜索范围，或降低可信度要求

### 问题 3：报告质量低
增加 `depth` 参数为 `deep`，或指定 `focus_areas`
